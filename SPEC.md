# dlang
*C without the weirdness*
## Specification
dlang is a compiled language that fills the role of C. 
### Type system
| Name | Data type | Size |
| --- | --- | --- |
| i32 | Number (signed) | 4 bytes |
| u32 | Number (unsigned) | 4 bytes |
| u8 | Byte (unsigned) | 1 byte |
